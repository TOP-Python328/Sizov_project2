         Тамагочи



    Описание:

Приложение, моделирующее жизнь и поведение питомца. 
Вид питомца можно выбрать из обширного списка.
Игрок сопровождает питомца на протяжении игровой жизни. За питомцем необходимо ухаживать, следить за его показателями. Также, с питомцем можно взаимодействовать.
После игровой смерти питомца можно завести нового.



    Интерфейс:

Графический интерфейс пользователя (GUI)



    Функциональность:


1. Главное меню
    
    а) в главном меню осуществляется выбор вида нового питомца


2. Сохранение и загрузка
    
    а) игрок может одновременно играть с одним питомцем
    
    б) текущее состояние питомца может быть сохранено для последующей загрузки


3. Параметры
    
    а) у питомца каждого вида есть базовый набор параметров: здоровье, настроение, ...
    
    б) питомцы некоторых видов обладают дополнительными параметрами
    
    в) значениями параметров являются вещественные числа из диапазона, определённого для каждого параметра
    
    г) во время работы приложения все параметры питомца обновляются в фоновом режиме ... раз за 1 ИД
    
    д) пересчёт параметров после повторного запуска приложения происходит во время загрузки существующего питомца 
    
    е) значения и изменение одних параметров влияет на изменение других


4. Действия игрока
    
    а) для питомца каждого вида определён базовый набор действий, совершаемых игроком
    
    б) для питомцев некоторых видов определены дополнительные действия
    
    в) действия игрока всегда влекут за собой разовое изменение значений некоторых параметров питомца


5. Активности питомцев
    
    а) для питомца каждого вида определён набор активностей, совершаемых питомцем во время работы приложения при отсутствии действий со стороны игрока в течении ... мин реального времени

    б) некоторые активности питомцев могут повлечь за собой разовое изменение значений некоторых параметров питомца


6. Время
    
    а) в качестве единицы измерения предлагается ИД (игровой день): в ИД измеряется время жизни питомцев
    
    б) во время работы приложения масштабирование времени устанавливается в соотношении 1 ИД к ... мин реального времени
    
    в) для пересчёта параметров питомца после повторного запуска приложения и загрузки существующего питомца масштабирование времени устанавливается в соотношении 1 ИД к ... ч реального времени, прошедшего между последним сохранением и текущей загрузкой


7. Возраст
    
    а) для питомцев разных видов может быть установлено несколько возрастных периодов - этапов развития
    
    б) для каждого возрастного периода устанавливается собственный диапазон для значений каждого параметра, собственный набор действий игрока и собственный набор активностей питомца
